# Arduino Countdown System

A simple Arduino project that combines a 7-segment display and a buzzer to create an interactive countdown. The display counts from **9 to 0**, with a short beep at each step and a longer tone when the countdown is complete.

## Features

* Countdown from 9 to 0
* 7-segment display output
* Buzzer sound feedback
* Serial Monitor updates

## Components

* Arduino Uno
* 7-segment display
* Passive buzzer
* Breadboard
* Jumper wires
* Resistors

## Pin Connections

* 7-segment display: Pins **2, 3, 4, 5, 6, 7, 9**
* Buzzer: Pin **8**

This project demonstrates the use of arrays, loops, functions, and digital output control to coordinate multiple hardware components.
